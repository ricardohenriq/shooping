FUNCIONALIDADES ATIVAS (FUNCIONALIDADES QUE SÃO FORTEMENTE LIGADAS A UMA ENTIDADE)
	
##################################################################################################

* FUNCIONALIDADES RELATIVAS A "USERS"

- USUÁRIO SE CADASTRAR (SOMENTE USUÁRIOS COMUNS (LOJISTAS E ADMIN CADASTRADOS PESSOALMENTE)) "POPUP".
- USUÁRIO FAZER PESQUISAS POR PRODUTO (LIVRE, NÃO NECESSITA LOGIN) "NOVA PÁGINA".
- USUÁRIO EDITAR SEU PERFIL (SOMENTE USUÁRIO DONO DA CONTA LOGADO OU ADMIN) "POPUP".
- USUÁRIO VISUALIZAR SEU PERFIL (SOMENTE USUÁRIO DONO DA CONTA LOGADO OU ADMIN) "NOVA PÁGINA".

* FUNCIONALIDADES RELATIVAS A "BANNERS"

- USUÁRIO VER QUAIS BANNERS ELE POSSUI NO SITE (SOMENTE USUÁRIO DONO DA CONTA LOGADO OU ADMIN) "NOVA PÁGINA".
- USUÁRIO VER DETALHES DE UM BANNER (SOMENTE USUÁRIO DONO DA CONTA LOGADO OU ADMIN) "POPUP".
- USUÁRIO EDITAR SEU BANNERS (SOMENTE USUÁRIO DONO DA CONTA LOGADO OU ADMIN) "POPUP".
- USUÁRIO ADICIONAR UM NOVO BANNER (SOMENTE USUÁRIO DONO DA CONTA LOGADO OU ADMIN) "POPUP".
- USUÁRIO DELETAR UM DETERMINADO BANNER (SOMENTE USUÁRIO DONO DA CONTA LOGADO OU ADMIN) "POPUP".

* FUNCIONALIDADES RELATIVAS A "BANNER_TYPES"

- USUÁRIO VER QUAIS BANNER_TYPES ELE ADICIONOU NO SITE (SOMENTE USUÁRIO ADMIN (VERIFICA SOMENTE 
	OS INSERIDOS POR ELE)) "NOVA PÁGINA".
- USUÁRIO VER TODOS OS REGISTROS DA TABELA "banner_types" (SOMENTE USUÁRIO ADMIN) "NOVA PÁGINA".
- USUÁRIO VER DETALHES DE UM BANNER_TYPES (SOMENTE USUÁRIO ADMIN) "POPUP".
- USUÁRIO ADICIONAR UM NOVO BANNER_TYPES (SOMENTE USUÁRIO ADMIN) "POPUP".
- USUÁRIO DELETAR UM DETERMINADO BANNER_TYPES (SOMENTE USUÁRIO ADMIN) "POPUP".
- USUÁRIO VER QUAIS OS "BANNERS" RELACIONADOS A UM DETERMINADO "BANNER_TYPES" (SOMENTE USUÁRIO ADMIN) "NOVA PÁGINA".
